# 直播间看过人数 (WATCHED_CHANGE)

当前直播历史观众数量，可替代人气。

## 下发条件

看过人数更新后。

## 数据包正文

数据包的正文，前19字节的信息未知:

```plain
          00 01 02 03 04 05 06 07 08 09 0A 0B 0C 0D 0E 0F

00000001: 8b 38 80 00 00 00 72 00 10 00 00 00 00 00 05 00  .8....r.........
00000002: 00 00 00                                         ...
```

## JSON消息

根对象:

| 字段 | 类型 | 内容             | 备注 |
| ---- | ---- | ---------------- | ---- |
| cmd  | str  | `WATCHED_CHANGE` |      |
| data | obj  | 信息本体         |      |

`data` 对象:

| 字段       | 类型 | 内容          | 备注 |
| ---------- | ---- | ------------- | ---- |
| num        | num  | 看过人数      |      |
| text_small | str  | 显示文字 (小) |      |
| text_large | str  | 显示文字 (大) |      |
